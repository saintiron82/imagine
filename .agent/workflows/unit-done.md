# 유닛 회고 및 완료

테스트 통과 후 회고를 작성하고 유닛을 완료 처리합니다.

## 실행 단계

### 1. 회고 작성
1. 다음 항목을 기록합니다:
   - **잘된 점**: 효과적이었던 접근법
   - **개선점**: 다음에 주의할 사항
   - **소요 시간**: 대략적인 작업 시간

### 2. task.md 업데이트
2. 현재 유닛의 체크박스를 `[x]`로 변경

### 3. 다음 유닛 확인
3. 다음 진행할 유닛 ID 확인
4. 의존성 충족 여부 확인

### 4. 커밋 포인트 (선택)
5. 필요시 변경사항 Git 커밋:
```powershell
git add .
git commit -m "Complete U-XXX: [유닛명]"
```

### 5. 다음 유닛 시작
6. `/unit-start` 실행하여 다음 유닛 진행
