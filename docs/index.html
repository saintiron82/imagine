Core Features

One search query. Three AI axes. Instant results across your entire asset library.

Triaxis Search

Visual similarity (VV) + semantic meaning (MV) + keyword matching (FTS) combined with Reciprocal Rank Fusion for unmatched search accuracy.

AI Vision Analysis

Qwen3-VL automatically generates captions, tags, and classifications. 2-stage pipeline: fast classify then detailed analysis.

Desktop + Web

Electron app for local use, or enable Server mode and let your team search from any browser. Same React UI, two modes.

Distributed Workers

Process files across multiple GPU machines in parallel. Smart prefetch pool, phased batching, and auto model unloading.
